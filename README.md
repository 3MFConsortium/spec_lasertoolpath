# 3D Manufacturing Format - Toolpath Extension

This repository contains the Toolpath Extension that extends the 3MF 3D Manufacturing Format Core specification.

This extension provides support for representing build-ready toolpath data within 3MF files. It defines model-level toolpath resources, process profiles, per-layer toolpath parts, planar and multi-axis geometry, laser-source metadata, synchronization, and extension points for machine-specific process information.

See the 3MF examples repository for 3MF toolpath sample files.
